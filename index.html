  • Как добавить иконку при компиляции в Code::Blocks? Язык программирования Си
    •
  • 1. Вам потребуется собственно сама иконка. Найти ее можно в сети или нарисовать самостоятельно. Расширение должно быть обязательно ico. Сохраняем ее в директорию рядом с вашим кодом. Вот файл main.c. Называем например icon.ico 2. В среде CodeBlocks в открытом проекте добавляем Файл - Новый - Пустой файл и называем его icon.rc. Это - файл скрипта ресурсов компилятора. На самом деле это текстовый файл, который включает в себя ссылки на ресурсы (такие как изображения, иконки и прочие компоненты, необходимые для пользовательского интерфейса. Найдем данный файл в папке Resources. 3. Прописываем в нем следующую строку, MAINICON ICON "icon.ico" где MAINICON это идентификатор, а icon.ico имя иконки. Сохраняем файл и компилируем. В папке bin расположен бинарник с добавленной иконкой. Что происходит под капотом можно увидеть в логах сборки. Файл rc компилируется программой windres в объектный файл. И после компилятор gcc собирает exe из объектных файлов, в том числе и icon.res

    •
  • Сегодня хочу коснутся темы генерации звука средствами языка Си. Для этих целей служит функция Beep. Она не является стандартной функцией языка Си, а включена в заголовочный файл utilapiset.h и работает с Windows API. Изначально предназначалась для воспроизведения простых однотонных звуков через встроенный динамик компьютера, но начиная с Windows 7 была переписана для воспроизведения звуковых сигналов синусоидальной формы с использованием API звуковой карты. Функция Beep принимает два параметра: частоту звука в герцах (37..32767) и длительность в миллисекундах.
